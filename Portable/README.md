## 🇬🇧 English

**Alpha Commander** is a fast and lightweight tool to convert PNG image sequences with transparency into WebM videos with an Alpha channel (true transparency).

Specially designed to preserve transparency without quality loss and optimize animations for the web or interfaces where dynamic backgrounds are key. It is **ideal for PNG sequences exported from Adobe After Effects** or other motion graphics software, allowing you to bring those animations directly to the web with an extremely low file size and a perfect alpha channel.

### ✨ Key Features

- **PNG to WebM (Alpha):** Automatically converts folders of PNG sequences to WebM videos maintaining a perfect alpha channel.
- **Boomerang Effect (Back and forth loop):** Enable the boomerang mode to play your animation from the first frame to the last and then in reverse, ideal for creating seamless infinite loops.
- **Preview & Alpha Verification:** Includes a built-in player to preview your WebM videos. You can dynamically change the background color (checkerboard, gradients, solid colors) to ensure the transparency is real and perfect.
- **Multi-language:** Clean interface available in English and Spanish.
- **100% Portable (All-in-One):** Everything comes in a single `.exe` file. No complicated installations and **no external dependencies**. The conversion engine (FFmpeg) is embedded internally and works completely invisibly.

### 📦 Download & Source Code

In this repository you will find:
1. **Portable Executable:** Ready to download and use immediately. No installation required.
2. **Source Code:** All the code is available so you can compile, study, or modify the application (built with Rust, Tauri, and Vanilla HTML/CSS/JS).

### 🚀 Quick Start (Portable)

1. **Run:** Open `AlphaCommander.exe`. No installation or extra programs required!
2. **Select Input/Output:** Choose the folder containing your PNG sequence (e.g. exported from After Effects) and where you want to save the final WebM.
3. **Configure:** Choose the name, FPS (12 to 60 fps) and enable the Boomerang mode if you want a seamless loop.
4. **Convert:** Click on "Convert to WebM".
5. **Verify:** Switch to the Preview tab, load your new WebM and verify the transparency by changing the backgrounds.

## 🇪🇸 Español

**Alpha Commander** es una herramienta rápida y ligera para convertir secuencias de imágenes PNG con transparencia en videos WebM con canal Alpha (transparencia real).

Diseñada especialmente para mantener la transparencia sin pérdida de calidad y optimizar animaciones para la web o interfaces donde los fondos dinámicos son clave. Es **ideal para secuencias de PNGs exportadas desde Adobe After Effects** u otros programas de motion graphics, permitiendo llevar esas animaciones directamente a la web con un peso súper ligero y sin perder el canal alpha.

### ✨ Características Principales

- **PNG a WebM (Alpha):** Convierte de manera automática carpetas con secuencias PNG a videos WebM manteniendo un canal alpha perfecto.
- **Efecto Boomerang (Loop ida y vuelta):** Activa el modo boomerang para reproducir tu animación del primer frame al último y luego en reversa, ideal para crear bucles infinitos sin cortes bruscos.
- **Preview y Verificación de Alpha:** Incluye un reproductor incorporado para previsualizar los videos WebM. Permite cambiar dinámicamente el color de fondo (checkerboard, gradientes, colores sólidos) para asegurar que la transparencia es real y perfecta.
- **Multi-idioma:** Interfaz limpia disponible en Español e Inglés.
- **100% Portable (All-in-One):** Todo viene en un solo archivo `.exe`. Sin instalaciones complicadas y **sin dependencias externas**. El motor de conversión (FFmpeg) está embebido internamente y funciona de forma totalmente invisible.

### 📦 Descarga y Código

En este repositorio encontrarás:
1. **Ejecutable Portable:** Listo para descargar y usar inmediatamente. No requiere instalación.
2. **Código Fuente:** Todo el código disponible para que puedas compilar, estudiar o modificar la aplicación (construida con Rust, Tauri, y HTML/CSS/JS Vanilla).

### 🚀 Uso Rápido (Portable)

1. **Ejecutar:** Abrí `AlphaCommander.exe`. ¡No necesita instalación ni programas adicionales!
2. **Seleccionar Entrada/Salida:** Elegí la carpeta donde tenés tu secuencia de PNGs (ej. exportada de After Effects) y dónde querés guardar el WebM final.

4. **Convertir:** Hacé click en "Convertir a WebM".
